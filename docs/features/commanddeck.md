# Feature: Command Deck

**Page:** `index.html`
**Module:** `js/modules/command-deck.js`

## Purpose

The Command Deck is the application's home page and central dashboard. It provides an at-a-glance view of the user's latest health and wellness metrics, quick navigation to other modules, backup management, and access to recent log history.

## User Workflow

1. Open USS TJR. The Command Deck loads as the default page.
2. The dashboard grid displays the latest Mood, Energy, Pain, and Stress scores from the most recent Captain's Log.
3. The current stardate is displayed in the Command Status section.
4. The user navigates to **Captain's Log** or **Medical Bay** using the navigation buttons.
5. After saving a log in either module, the user returns to the Command Deck to see updated metrics.
6. The user can search recent logs using the search field in the Recent Logs section.
7. The user can export a backup using the Backup section controls.

## Data Displayed

| Element | Source |
|---------|--------|
| Mood score | `usstjr-latest-captains-log` → `mood` |
| Energy score | `usstjr-latest-captains-log` → `energy` |
| Pain score | `usstjr-latest-captains-log` → `pain` |
| Stress score | `usstjr-latest-captains-log` → `stress` |
| Current stardate | Generated by `js/core/dates.js` on page load |
| Recent logs list | `usstjr-captains-log-history` (up to 20 entries) |

## Key Actions

| Action | Description |
|--------|-------------|
| Navigate to Captain's Log | Opens `captains-log.html` |
| Navigate to Medical Bay | Opens `medical-bay.html` |
| Export Backup | Downloads all data as a plain `.json` file |
| Import Backup | Restores data from a selected `.json` file |
| Export Encrypted Backup | Downloads AES-GCM encrypted `.enc.json` file |
| Import Encrypted Backup | Restores data from encrypted file with passphrase |
| Search Logs | Filters the recent logs list by stardate, date, text, or metric values |
| View Log Entry | Clicks a history entry to open it in Captain's Log |
| Delete Log Entry | Removes a single entry from history (with confirmation) |
| Clear All History | Removes all entries from history (with confirmation) |
